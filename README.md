# tima.dev — Портфолио

Минималистичное портфолио IT-специалиста и веб-разработчика. Статический сайт для GitHub Pages.

## Локальный просмотр

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx serve .
```

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub (например `profolio` или `tima.dev`)
2. Загрузите файлы в ветку `main`
3. **Settings → Pages → Source**: Deploy from branch `main`, folder `/ (root)`
4. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`

## Что поменять под себя

- **Почта** в `index.html` — замените `timur@example.com` на реальную
- **Сертификаты** — обновите названия и описания в секции `#certs`
- **Фото** — замените URL аватара GitHub на своё фото в `about__photo`
