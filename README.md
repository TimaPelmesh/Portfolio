# tima.dev — Портфолио

Минималистичное портфолио IT-специалиста и веб-разработчика. Статический сайт для GitHub Pages.

## Локальный просмотр

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx serve .
```

## Деплой на GitHub Pages

1. Репозиторий: [TimaPelmesh/Portfolio](https://github.com/TimaPelmesh/Portfolio)
2. **Settings → Pages → Source**: Deploy from branch `main`, folder `/ (root)`
3. Сайт: `https://timapelmesh.github.io/Portfolio/`

## Что поменять под себя

- **Фото** — положите файл `photo.jpg` в корень репозитория (рядом с `index.html`)
- **Почта** в `index.html` — замените `timur@example.com` на реальную
- **Сертификаты** — добавьте карточки в секцию `#certs` (сейчас стоит заглушка «Скоро»)

## Стек

HTML · CSS · Vanilla JavaScript — без фреймворков, быстрая загрузка.
